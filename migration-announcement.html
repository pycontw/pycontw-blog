PyCon Taiwan 從 Blogger 搬到 GitHub 上囉～

Posted on Mon 09 January 2023 in announcement

PyCon Taiwan 的部落格正式由 Blogger 搬到 GitHub / https://conf.python.tw 上囉！
新的部落格除了是使用由 Python 攥寫的 pelican 來產生外，也希望能透過開放原始碼的方式，讓更多的人能一起來讓 PyCon Taiwan 的部落格內容更加豐富！
接下來，也要請大家繼續支持 PyCon Taiwan！

We migrate our existing PyCon Taiwan blog from Blogger to GitHub / https://conf.python.tw.
Our new blog is generated by pelican, a Python-written static site generator, and open-sourced.
By doing so, we hope more people can contribute and post content.
