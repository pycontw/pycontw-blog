<p>我們在<a href="https://pycontw.blogspot.com/2021/03/python.html" target="_blank">前一篇</a>提到為什麼要加入 Python 軟體基金會（ Python Software Foundation，PSF ）的基本會員，接下來我們要來說明如何提出申請、成為會員。</p><p>大致上的流程是先在 https://www.python.org 建立帳號，然後透過這個代表本人的帳號簽署行為準則（ Code of Conduct ）即可，是不是很簡單呢？下面是實際上每個步驟的截圖提供給大家參考。</p><p><br /></p><p><br /></p><h1 style="text-align: left;">申請成為 PSF Basic Membership</h1><div><br /></div><h2 style="text-align: left;">註冊 www.python.org 帳號</h2><div><br /></div><div>首先先在 https://www.python.org 建立帳號。你可以直接點擊 https://www.python.org/accounts/signup/ 連到註冊畫面：</div><div><br /></div><div class="separator" style="clear: both;
