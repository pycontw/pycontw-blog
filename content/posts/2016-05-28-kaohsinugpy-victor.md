<div style="text-align: center; margin: 3em 0 3em 0;">
    <strong><blockquote>南部地區的 Python 愛好者們， 你們 Kaohsiung.py 了嗎？</blockquote></strong>
</div>

<p>
隨著 Python 的流行，台灣各地都有在地化的 Python 社群，<br />
位於南部地區的 Python Programmer，千萬不能錯過位於高雄的 Kaohsiung.py。<br />
</p>
<p>
Kaohsiung.py 分享主題涵蓋許多領域，<br />
從 Web 開發，資料處理，以及時下最熱門的深度學習與機器學習，<br />
分享的題目內容非常多元。
</p>
<p>
主辦 Victor 會不定期邀請來自台灣各地的講者，<br />
除了讓題目更加豐富，高雄地區的朋友們，能有更多機會接觸到台灣各地的 Python 好手。
</p>

<br />
<div class="separator" style="clear: both; text-align: center;"><img border="0" src="https://2.bp.blogspot.com/-FVBoqlXKWIU/V0h9HRQREwI/AAAAAAAAD2c/wPo2amXR2gkEsvpaLGD9A1sO_We7stEFACLcB/s1600/kaohsiung2.jpg" width="80%" /></div>
<br />

<p>
Kaohsiung.py 每個月會有一次技術性的分享，地點位於高雄文藻外語大學，<br />
聚會人數都在二十人以上，發展非常活躍。
</p>

<p>
想知道更多嗎？趕快加入 <a href="https://www.meetup.com/Kaohsiung-Python-Meetup/">Kaohsiung.py Meetup</a>，參與下一次的主題分享！
</p>
